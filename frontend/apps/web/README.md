# Grocery Web App

## Run

From the repo root:
- `cd frontend`
- `pnpm run dev:web`
- `pnpm run build:web`
- `pnpm run lint:web`

From this folder:
- `pnpm run dev`
- `pnpm run build`
- `pnpm run lint`

## Environments

Vite sets the mode used by `src/config`:
- Dev: `pnpm run dev`
- Test: `pnpm run test`
- Prod build: `pnpm run build`
